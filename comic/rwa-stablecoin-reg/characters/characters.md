# Character Definitions - RWA 稳定币监管知识漫画

**Style**: ligne-claire（清晰线描，欧洲漫画风格）
**Art Direction**: 干净线条、平涂色彩、人物特征鲜明易识别

---

## Character 1: 小白（读者视角）

**Role**: Protagonist（主角，代表普通投资者）
**Age**: 28岁

**Appearance**:
- Face shape: 圆脸，略带稚气
- Hair: 黑色短发，略乱，刘海搭在额头
- Eyes: 大眼，黑色瞳孔，常带疑惑表情
- Build: 中等身材，瘦长形
- Distinguishing features: 戴一副简约黑框眼镜，额头有撮翘起的头发

**Costume**:
- Default outfit: 白色T恤 + 牛仔衬衫（敞开），休闲裤
- Color palette: 白色（#FFFFFF）、牛仔蓝（#6C9BD2）、黑色眼镜框（#000000）
- Accessories: 智能手机（手中常拿）、双肩包

**Expression Range**:
- Neutral: 平静，眼镜后的眼睛略带好奇
- Happy/Excited: 眼睛睁大，嘴角上扬，双手握拳
- Thinking/Confused: 眼镜下滑，眉头微皱，手挠头
- Determined: 眼镜推回鼻梁，下巴微抬

---

## Character 2: 监管专家 李教授（讲解者）

**Role**: Mentor（导师，知识讲解者）
**Age**: 45岁

**Appearance**:
- Face shape: 方脸，轮廓清晰
- Hair: 灰黑短发，整齐向后梳，鬓角微白
- Eyes: 细长的眼睛，眼神锐利，戴无框老花镜（挂胸前）
- Build: 中等偏壮，站姿挺拔
- Distinguishing features: 下巴有一粒痣，常带自信微笑

**Costume**:
- Default outfit: 深蓝色西装外套，浅蓝衬衫，不系领带
- Color palette: 深蓝（#1B3A6B）、浅蓝（#B8D4E3）、灰色西装裤（#808080）
- Accessories: 胸口袋插一支钢笔、手腕戴智能手表

**Expression Range**:
- Neutral: 微笑，双手交叉在胸前，自信
- Happy/Excited: 眼睛发亮，手在空中比划
- Thinking/Confused: 抬头思考，手指轻敲下巴
- Determined: 前倾身体，手指轻点桌面，强调语气

---

## Character 3: 银行家 王行长（反对派代表）

**Role**: Antagonist（对手方，代表传统银行业立场）
**Age**: 50岁

**Appearance**:
- Face shape: 宽脸，略带富态
- Hair: 深棕色短发，整齐梳理，发际线略退
- Eyes: 小眼睛，常半眯，算计表情
- Build: 壮实，略胖
- Distinguishing features: 戴名贵腕表（表面偶尔反光）、西装永远扣着扣子

**Costume**:
- Default outfit: 深灰色三件套西装，白衬衫，红色领带
- Color palette: 深灰（#36454F）、白（#FFFFFF）、暗红（#8B0000）
- Accessories: 金色腕表（#FFD700）、公文包

**Expression Range**:
- Neutral: 假笑，双手交叉在腹部
- Happy/Excited: 罕见，只在看到利润数字时
- Thinking/Confused: 皱眉，手指轻敲桌面
- Angry/Worried: 脸红，领带松开，手摆文件

---

## 场景/符号角色

### 符号：稳定币（USDC 形象）
- 圆形金币造型，蓝色底色，上有 "USDC" 字样
- 表情：开心（合规时）、担心（监管不确定时）
- 位置：常在小白手中或作为背景符号出现

### 符号：CLARITY Act / GENIUS Act（法案形象）
- 用「巨大的法典/石碑」形象表示
- 石碑上有法案名称，散发金光（通过时）或灰雾（未通过时）
