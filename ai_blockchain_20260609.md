# 欧洲的静默铺管：一场没有镁光灯的金融基建革命

## 【AI×区块链】Money20/20 Europe 揭示的三层管线，正在让 AI Agent 支付在欧洲先跑起来

### 文章解读

6月2-4日，Money20/20 Europe 在阿姆斯特丹落幕。7400人参会，三条主线同时浮出水面：AI Agent 支付进入生产环境、37家银行联手发行欧元稳定币、开放银行支付框架直接挑战 Visa/Mastercard 的双头垄断。

这三件事放在一起看，结论很清楚——欧洲正在悄然完成"AI Agent 能用什么钱、走什么通道、受什么监管"的三层基建回答。这不是一个国家的战略宣示，而是一整片大陆的管道对接。

对比之前我们讨论过的日本"三段解锁"和美国的监管转型，欧洲走的是第三条路：不喊口号，只接管道。

---

### 一、为什么关注这个话题

过去三个月，我们讨论了日本的 AI+区块链战略、香港的稳定币牌照、迪拜的房地产代币化、美国 SEC 的战略转身。这些故事都有一个共同缺口：**AI Agent 到底怎么付钱？**

6月2日，答案在欧洲出现——Worldline、ING、Mastercard 三方完成了欧洲首笔生产环境下的端到端 AI Agent 支付交易。不是概念验证，不是沙盒，是真实生产环境。

---

### 二、行业现状

Money20/20 Europe 2026 释放了三个具体信号：

**第一层：支付管道就绪。** Worldline+ING+Mastercard 的 AI Agent 支付在荷兰和比利时双市场同步运行。Agent 负责筛选推荐，消费者明确批准后完成支付，交易携带 Agent 标识符，发卡行全程可控。Mastercard 封装为 Agent Pay 产品。

**第二层：货币管道就绪。** Qivalis——37 家欧洲银行联盟（ING、BNP Paribas、Deutsche Bank、UniCredit、Intesa Sanpaolo 等），覆盖 15 国——在荷兰央行监管下发行 MiCA 合规欧元稳定币，2026 年底上线。欧洲银行用自己的资产负债表和客户网络发行，Société Générale 的 EURCV 两年仅 $1.22 亿，而 Qivalis 从第一天起就把存量客户变成潜在用户。

**第三层：监管管道就绪。** MiCA 全面执行，CASPs 不持牌即下架。看似收紧，实为清场——为合规玩家腾出确定性跑道。

---

### 三、深度分析

把三层管道叠加起来，欧洲构建的是一套完整的"AI Agent Commerce"基础设施：

- **AI Agent 能用什么钱？** → Qivalis 欧元稳定币（银行级）、GBPA 英镑稳定币（Clear Junction+Agant）、甚至 MoneyGram 的 MGUSD（Stellar 链上）
- **AI Agent 走什么通道？** → Mastercard Agent Pay（卡网络）、UKPI 可变递归支付（开放银行，直接绕开卡组织）
- **AI Agent 受什么监管？** → MiCA（稳定币发行）+ EU AI Act（Agent 行为合规）+ DNB/ECB（央行监管）

三个引人注目的细节：

**细节一：银行联盟的速度。** Qivalis 2025年9月成立时仅9家银行，2026年5月一次性新增25家至37家。欧洲银行对美元稳定币主导跨境结算的焦虑是真实的——欧元稳定币月交易量从1月 $6900 万飙至5月 $7.77 亿，增速超10倍。

**细节二：Mastercard 的两面下注。** 3月以 $18 亿收购稳定币基础设施商 BVNK，6月推出 Agent Pay——同一家公司在同时铺设"稳定币清算管道"和"AI Agent 支付管道"。而 UKPI 的开放银行 VRP 框架（30+ 机构参与）正在另一端绕开 Mastercard 自己的卡网络。

**细节三：ECB 的缺席反而加速了市场。** 数字欧元最早2029年推出，试点仅5000-10000用户。Qivalis 抓住了这个窗口：在 ECB 走完官僚流程前，先用银行联盟占领机构市场。

---

### 四、投资视角

三个直接机会方向：

1. **欧洲稳定币基础设施**：Qivalis 底层技术提供商、MiCA 合规审计、欧元稳定币做市和流动性层——2026年底发币前后确定性最高。
2. **Mastercard Agent Pay 生态**：首个规模化 AI Agent 支付标准，商户接入工具、风控层、身份验证中间件有早期红利。
3. **开放银行 VRP 赛道**：TrueLayer "Bank on File"已落地 Trading 212 和 IG Group，绕开卡网络做 AI Agent 支付在技术层面已跑通。

风险：Qivalis 的37家银行联盟可能重蹈 Libra/Diem 覆辙——成员越多，决策越慢。稳定币市场"很少奖励第二名入场者"。

---

### 五、消费者/用户视角

对普通用户来说，三层管道落地的直接体验：未来你的银行 App 里会出现一个 AI Agent 助手，你设定预算和偏好，它帮你比价、下单、支付——不必每次都手动输入卡号、验证短信。ING 已经在做这件事。

但消费者控制权是设计红线：ING+Worldline 方案要求用户给出明确批准，Agent 不会完全自主支付。与日本 MUFG 的委托购买模型不同，欧洲选了更保守的路径，也因此更容易获得监管和用户信任。

---

### 六、结论：机会点

欧洲模式的核心竞争力不是技术创新，而是**三层基建的同步完成**。美国有最强的 AI 公司但缺统一监管框架，亚洲有最活跃的加密市场但 Agent 支付标准还在纸面上。

机会不在"押注欧洲AI代币"：
1. **合规基础设施出口**：MiCA 框架下的稳定币合规方案可输出到其他地区
2. **AI Agent 支付中间件**：Agent Pay 和 VRP 的接入层、风控层、审计层
3. **跨境监管互认**：欧盟框架与香港、新加坡、日本的对接商机

欧洲在这场竞赛中的策略是：不喊口号，只接管道。等其他人吵完，管道已经通了。
