### rwa_publish_wp.py 重大修复
- **WP REST API 对大 payload 有 Cloudflare/代理问题** — >~7.5KB 的 POST 会返回 504/000
- **解决方案：改用 XML-RPC** (`xmlrpc.php`) — 大 payload 正常传输
- `xmlrpc.client.dumps(params_tuple, methodname="wp.newPost")` — params 需包含 `(blog_id=1, username, password, content_struct)`
- Python `requests` 库通过代理也有 chunked encoding/超时问题
- curl `--data-binary @file` / `-d @-` 对大方体通过代理也不稳定
- 标题提取已支持 `#` 和 `##` 两种格式（早晚报都用 `#` 单级标题）
