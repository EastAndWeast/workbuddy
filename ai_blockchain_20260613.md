# 三层一体，一周成局：AI Agent 经济的基础设施拐点

## 【AI×区块链 周报】2026年6月8日-6月13日

### 周度解读

这一周，AI Agent 经济的三层基础设施几乎同时就位：MetaMask Agent Wallet 给了 Agent 钱包（6/8），Mastercard Agent Pay for Machines 给了 Agent 支付网络（6/10），日本 FIEA 修正案给了 Agent 合法交易标的（6/11）。当传统金融巨头和加密原生协议在同一条跑道上对齐，赛道的拐点已经到了。另一面，BTC 经历创纪录的 13 天 $44 亿 ETF 净流出，DeFi 年内损失突破 $8 亿——资金正从"赌方向"转向"买管道"，但安全护栏的建设速度远慢于 Agent 能力的释放速度。

### 🔥 本周 TOP 3 事件

**1. MetaMask Agent Wallet + Mastercard Agent Pay：Agent 的"银行账户"和"Visa 网络"同周上线**

6 月 8 日，Consensys 推出 MetaMask Agent Wallet——200 个早鸟名额，AI Agent 在用户保留私钥控制权的前提下自主执行 DeFi 交易（Swap、永续合约、预测市场）。两天后，Mastercard 发布 Agent Pay for Machines，30+ 合作伙伴（Ripple、Coinbase、Stripe）让 Agent 实现跨卡、跨银行、跨稳定币的自主结算。钱包层+支付层，一周内同时落地。

- 为什么重要：Agent 不再只是"能聊天"，它有钱包、有支付网络，从对话工具变成经济参与者。传统金融和加密原生在 Agent 目标上形成事实分工。
- 来源：[MetaMask Agent Wallet](https://www.coingabbar.com/en/crypto-currency-news/metamask-agent-wallet-ai-defi-trading-early-access-june-2026) | [Mastercard Agent Pay](https://www.mastercard.com/us/en/news-and-trends/press/2026/june/mastercard-launches-agent-pay-for-machines.html)

**2. 日本 FIEA 修正案众议院通过：加密纳入证券法，20% 税率+ETF 通道**

6 月 11 日，日本众议院通过《金融商品交易法》修正案，105 种加密资产从支付工具重分类为金融商品。资本利得税从最高 55% 降至 20%，为 BTC 和 XRP ETF 打开法律通道——G7 首个将加密全面纳入证券监管框架的立法动作。税率降幅 63.6%，¥5 万亿加密存款的结构性流动性有望释放。

- 为什么重要：法律层是 Agent Commerce 的最后一块拼图——有了合法交易的商品，钱包和支付网络才有用武之地。
- 来源：[Japan FIEA Bill 2026](https://www.spotedcrypto.com/japan-crypto-fiea-bill-2026-tax-etf-pathway/) | [Japan Crypto Law ETF Path](https://www.cryptotimes.io/2026/06/11/japans-new-crypto-law-clears-the-path-for-bitcoin-and-xrp-etfs/)

**3. SEC 五年战略计划：AI 和区块链首次写入同一战略句式**

6 月 2 日发布的 SEC 2026-2030 战略计划草案，首次为数字资产设置独立目标章节，并将 AI 和区块链写入同一句话。3 月 SEC-CFTC MOU 结束管辖权战争，5 月 CLARITY 法案通过参议院银行委员会，6 月战略计划制度化——美国监管从"执法铁拳"转向"监管基建"。公共评论期至 7 月 2 日。

- 为什么重要：如果评论期后框架确定，将为 AI Agent + 链上金融铺设联邦级合规轨道。
- 来源：[SEC Strategic Plan](https://www.sec.gov/files/draft-strategic-plan-fy26-fy30.pdf) | [SEC Digital Assets Priority](https://cointelegraph.com/news/sec-makes-digital-assets-strategic-priority-through-2030)

### 📈 趋势观察

**资金在买管道，不买赌注。** BTC ETF 连续 13 天净流出 $44 亿，山寨币普跌 5-7%，但 AI Agent 基础设施（MetaMask、Mastercard、Amber A-MM）和 RWA 代币化赛道逆势吸金。BlackRock BUIDL 规模超 $23 亿，RWA 代币化资产同比增 600%，债券和货币市场基金贡献 $65 亿增量。市场信号明确：谁掌握 Agent 经济的"管道"——钱包、支付、合规发行——谁就掌握下一个周期的定价权。

**安全仍是最大暗角。** DeFi YTD 损失 $8.17 亿，6 月 10 日 Raydium 遭 $134 万漏洞攻击，CertiK 报告跨链桥损失>$3.28 亿。当 Agent 获得自主交易能力，安全护栏的建设速度远慢于 Agent 能力的释放速度——这是整个赛道最大的结构性风险。
